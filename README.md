# 運動社團WEB應用程式
  
## 應用程式簡要
這是一個可以創立運動社團並參與社團活動的WEB應用程式  
使用者可做為一般使用者或社團管理員登入  
一般使用者可以在此應用程式中查詢或加入社團  
查詢或參加社團所舉辦的活動  
也可以透過查詢找到社團發布的活動訊息, 文章  
社團管理員可以發布社團活動  
社團活動型式分為比賽, 課程或聚會  
也可發布最新訊息, 文章
